# Quy tắc: Không tự động push

Từ bây giờ, chỉ commit code local. KHÔNG gọi `git push` trừ khi người dùng yêu cầu rõ ràng.

**Lý do:** Project VocabularyApp bị tự động push nhiều lần không cần thiết, gây phiền.
